**Deskripsi Proyek** :
Proyek ini merupakan Sistem Informasi dan Prediksi Dini Bencana Banjir Kota Palu yang dirancang untuk memantau kondisi cuaca, menampilkan data historis banjir, serta memberikan prediksi tingkat risiko banjir pada tiap kecamatan. Sistem mengintegrasikan visualisasi data, peta interaktif, dan model prediksi berbasis aturan untuk membantu pengguna memahami pola curah hujan, tingkat kerentanan wilayah, dan potensi terjadinya banjir.

Aplikasi ini memuat beberapa modul utama, yaitu: Dashboard statistik, tabel data historis, diagram analisis, peta sebaran kejadian, dan fitur prediksi yang memungkinkan pengguna memasukkan parameter cuaca seperti curah hujan, kelembapan, suhu, dan tinggi muka air. Semua komponen bekerja secara terintegrasi untuk memberikan gambaran menyeluruh mengenai situasi banjir di Kota Palu serta mendukung pengambilan keputusan secara cepat dan tepat.
